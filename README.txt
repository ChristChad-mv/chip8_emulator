État du Projet
Nous avons atteint les objectifs demandés et validé toutes les ROMs de test :

✅ Logos
✅ Corax+
✅ Flags
✅ Quirks
✅ Keypad
✅ Beep
✅ Un jeu complet
 - Jeu de Paddles
 - Jeu de Ping Pong
 
Le programme se quitte proprement lorsqu'on appuie sur la croix.

Structuration et Organisation du Code
Les structures définies correspondent à des concepts existants.
Le code est réparti logiquement dans plusieurs fichiers :
src/ : fichiers sources .c
include/ : fichiers d'en-tête .h
Makefile : pour la compilation du projet

Qualité du Code
Le code est clair et bien indenté.

Des commentaires sont fournis pour expliquer chaque fonction et même des fichiers headers et sources.

Nous avons des fonctions de libération de mémoire. 

Pour chaque erreur, nous avons testé les retours. 
Pour d'autres fonctions, on retourne 0 si tout s'est bien passé et 1 le contraire.
